## Installation

Step 1 : Create a Virtual Environment

```bash
  python -m venv venv
```

Step 2 : Activate the Virtual Environment

```bash
  vnve\Scripts\Activate
```

Step 3 : Install the requirement file

```bash
  pip install -r requirements.txt
```

Step 4 : Run the file

```bash
  run document_similarity.py
```
